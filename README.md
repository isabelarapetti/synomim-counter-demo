# synonim-counter-demo

**Build API:**

run:

    cd server

**build without docker:**

    npm install
    npm run start

**or build with docker:**

    npm run docker:build
    npm run docker:run

---

**Build Angular WebApp:**

run:

     cd client/angular-client

**build without docker:**

    npm install
    npm run start

---

**Build DB**

    docker compose up
